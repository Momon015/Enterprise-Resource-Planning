{% load humanize %} {% load payment_tags %} {% comment %} Pending-sale review modal — loaded into #confirmBody via htmx, which auto-opens the global confirm modal (see main.html). WHY this exists: Confirm and Cancel used to sit on the list row, so a parked sale was posted blind — the row only showed a reference, a line count and a total. That is fine for a sale rung up a minute ago and useless for one parked three days ago with 25 lines on it. Confirming is the LAST reversible moment (afterwards the row locks and the only correction is a return), so the buttons moved here: you cannot post the sale without first being shown what you are posting. Cancel drills DOWN into the existing cancel modal by re-targeting #confirmBody — the same element this modal lives in, so it replaces this panel rather than stacking a second overlay. {% endcomment %}
