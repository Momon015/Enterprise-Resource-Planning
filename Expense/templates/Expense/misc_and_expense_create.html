Template Guide
What is a template?

A template is a saved expense you pay repeatedly — like rent or utilities. Instead of typing the details every time, you just check it when logging.

Checklist
  • Use a clear name (e.g. "Electricity", "Rent").
  • Set a realistic default amount.
  • Assign a category for better reporting.
